# Schematics (Litematica / cubical.xyz)

These files are **Minecraft Structure NBT** schematics (`.nbt`).

## Litematica Import
1. Copy `.nbt` files into:
   `.minecraft/schematics/`
2. In-game: Litematica → **Load Schematic** → select the `.nbt`
3. Place and rotate as needed
4. (Optional) Save as `.litematic` inside Minecraft

## cubical.xyz
Upload any `.nbt` to view, slice layers, and share.
