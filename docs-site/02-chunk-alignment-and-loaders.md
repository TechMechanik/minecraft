# Chunk Alignment & Loaders

## Alignment Rules
- Build with F3+G on
- Atrium shaft centered in a chunk
- Module interiors mostly within one chunk
- Structural borders may overlap

## Loader Strategy

### Primary Loader
- Exactly one
- Atrium center, Y=1
- Loads entire base vertically

### Secondary Loaders
Use only for:
- Farms
- Mob systems
- Extreme automation

Rules:
- One per vertical stack
- Centered
- Never on edges
