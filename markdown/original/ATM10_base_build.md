Below is a **clean, modular material system** for **ATM10**, using **current, commonly-available mod blocks** (Create, Ars Nouveau, Botania, Occultism, Quark, Supplementaries, Chipped, etc.).
Nothing here relies on vanilla-only aesthetics, and each module is visually distinct but cohesive.

No fluff. This is a **build spec** you can follow.

---

# GLOBAL DESIGN RULES (IMPORTANT)

**Module Size:** 17×17
**Interior Height:** 6 blocks (4-block windows + 2 structural layers)
**Theme:** Modern lodge meets arcane-industrial

All modules share:

* Same footprint
* Same tunnel system
* Same roof style (green roof or slab roof)
* Different **floor + wall palettes only**

---

# CONNECTING TUNNELS (ALL MODULES)

These must visually “belong everywhere”.

### Tunnel Size

* **5 wide × 5 tall**
* Flat ceiling for cables/pipes

### Tunnel Materials (STANDARDIZED)

| Element  | Block                                                     |
| -------- | --------------------------------------------------------- |
| Floor    | **Create → Andesite Casing**                              |
| Walls    | **Quark → Polished Tuff Bricks**                          |
| Ceiling  | **Dark Oak Slabs**                                        |
| Trim     | **Create → Brass Casing (corners only)**                  |
| Lighting | **Create → Brass Lanterns** or **Supplementaries Sconce** |

✔ Industrial + magical neutral
✔ Matches tech, magic, and nature builds
✔ Looks intentional, not temporary

# MODULE MATERIAL SPECIFICATIONS

---

## 1. CORE / STORAGE MODULE

**Purpose:** Storage, crafting, logistics hub

### Flooring

* **Create → Factory Floor (Dark)**
* Border: **Create → Andesite Casing**

### Walls

* **Chipped → Reinforced Stone Bricks**
* Window framing: **Create → Andesite Pillar**

### Accent

* **Framed Glass** (for massive windows)
* **Storage Drawers Trim** as wall detail

**Vibe:** Industrial foundation of the base

---

## 2. POWER MODULE (WATER-SIDE)

**Purpose:** Early → Mid power generation

### Flooring

* **Create → Factory Floor (Hazard / Yellow-Black)**

### Walls

* **Create → Copper Casing**
* Oxidize naturally or wax depending on style

### Ceiling

* **Create → Copper Shingles (Slabs)**

### Accent

* **Immersive Engineering → Treated Wood Fence**
* Exposed mechanical belts / shafts

**Vibe:** Mechanical dock power house

---

## 3. TECH / MACHINES MODULE

**Purpose:** Processing, automation, cables

### Flooring

* **Mekanism → Steel Casing**
* OR **Industrial Foregoing → Machine Frame Floor**

### Walls

* **Mekanism → Structural Glass**
* Framed with **Refined Storage → Quartz Iron**

### Accent

* **RS Cables in floor trenches**
* Glow Panels (Quark)

**Vibe:** Clean, high-tech lab

---

## 4. FARM / NATURE MODULE (FOREST-SIDE)

**Purpose:** Crops, trees, bees, Botany Pots

### Flooring

* **Botania → Livingwood Planks**
* Inset strips of **Moss Carpet**

### Walls

* **Botania → Livingrock Bricks**
* Alternating leaf blocks (vertical strips)

### Ceiling

* Fully glass or leaf lattice

### Accent

* **Botania → Mana Glass**
* Hanging planters (Supplementaries)

**Vibe:** Greenhouse meets elven ruin

---

## 5. ✨ MAGIC MODULE (NEW)

This is where ATM10 shines.

### Flooring (Choose One Style)

**Primary (Recommended):**

* **Ars Nouveau → Arcane Stone**
* Center rune ring using **Source Gem Blocks**

**Alternate (Occultism):**

* **Occultism → Chalk Rune Inlays** on polished stone

---

### Walls (Layered Look)

| Layer           | Material                              |
| --------------- | ------------------------------------- |
| Base            | **Ars Nouveau → Arcane Stone Bricks** |
| Vertical Inlays | **Botania → Runed Livingrock**        |
| Upper Trim      | **Chipped → Carved Deepslate**        |

---

### Windows

* **Ars Nouveau → Scrying Glass**
* Or **Botania → Mana Glass** for glowing edges

---

### Ceiling

* **Astral Sorcery-style star pattern** (even if AS isn’t used)
* Dark blocks + embedded glow crystals

---

### Accent Blocks

* **Ars Nouveau → Source Jar (embedded)**
* **Occultism → Spirit Fire Lamps**
* Floating bookshelves (Supplementaries)

**Vibe:** Arcane sanctum, not wizard shack
