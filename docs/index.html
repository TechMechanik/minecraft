Overview

This base design is intended for long-term ATM10 worlds.

Design Pillars

  • Modular: nothing blocks expansion
  • Deterministic: no guesswork mid-build
  • Chunk-safe: prevents desyncs and unload bugs
  • Discipline-separated: tech, magic, farming isolated
  • Vertical-first: scales upward and downward

The central atrium is the only global structure.

Everything else plugs into it.

Global Standards

Module Dimensions

  • Footprint: 17×17
  • Interior height: 6
  • Structural height per level: 8

Vertical Levels

Floors repeat every 8 blocks:

  • Ground: Y = 0
  • Upper floors: +8, +16, +24
  • Basements: -8, -16

Tunnel Standard

  • Size: 5×5
  • Flat ceiling for cables

Chunk Alignment & Loaders

Alignment Rules

  • Build with F3+G on
  • Atrium shaft centered in a chunk
  • Module interiors mostly within one chunk
  • Structural borders may overlap

Loader Strategy

Primary Loader

  • Exactly one
  • Atrium center, Y=1
  • Loads entire base vertically

Secondary Loaders

Use only for:

  • Farms
  • Mob systems
  • Extreme automation

Rules:

  • One per vertical stack
  • Centered
  • Never on edges

Basement Expansion

Philosophy

  • Ugly systems go down
  • Performance over looks

Basement Types

  1. Utilities
  2. Heavy machinery
  3. Dark rituals

Access

  • Atrium shaft only

Block Counts

Shared (Per Module)

  • Foundation: 289
  • Pillars: 28
  • Roof slabs: 136
  • Glass: 160

Use Litematica

Counts assume no interior machines.

Build Order

  1. Enable chunk borders
  2. Place atrium + shaft
  3. Add primary loader
  4. Build ground modules
  5. Add basements
  6. Add upper floors
  7. Add transport
